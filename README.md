## 快速开始

### 环境要求

- Node.js >= 22
- QQ 路线：Windows / macOS，需能访问 QQ 小程序本地资源文件
- 微信路线：Windows，需可用的微信 PC 调试环境

### 方式一：启动脚本（推荐）

**Windows：** 双击运行 `Windows_start.bat`

**macOS：**

```bash
chmod +x ./Mac_start.sh
./Mac_start.sh
```

按提示选择 `普通启动 / Lite 悬浮窗`，以及 `QQ / 微信` 路线，等待浏览器自动打开控制页。

默认控制页地址：http://127.0.0.1:8787/

### 方式二：命令行

```bash
# 安装依赖
npm install

# 启动 QQ 路线
npm run start -- --qq

# 启动微信路线
npm run start -- --wx
```
